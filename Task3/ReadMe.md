Читает строки из консоли пока не нажмут Ctrl+D

Разбивает каждую строку на отдельные слова

Выводит слова в том порядке, как они были в строке

Сортирует слова по алфавиту и выводит еще раз

Понимает специальные символы: |, &&, >>, << и другие

Правильно обрабатывает комбинации без пробелов: cat|sort&&ls

Всегда выбирает самые длинные специальные слова

Файлы программы
main.c - главная программа, читает строки и управляет процессом

list.h / list.c - работа со списком слов (добавление, сортировка, вывод)

tokenizer.h / tokenizer.c - разбор строки на отдельные слова

Makefile - автоматическая сборка программы

Сборка:
make

Запуск:
./program
